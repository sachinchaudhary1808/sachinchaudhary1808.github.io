# Sachin Chaudhary's Website

This repository hosts my minimalist personal website, designed for GitHub Pages.

## Features

- **Simple Design:** A clean and minimal layout, inspired by Linux users.
- **About Me:** A brief introduction showcasing my interests and passions.
- **Social Links:** Quick access to my GitHub, Twitter, and LinkedIn profiles.

## How to Use

1. Clone this repository:

   ```bash
   git clone https://github.com/sachinchaudhary1808/sachinchaudhary1808.github.io.git
   ```

2. Navigate to the repository and preview the website locally:

   ```bash
   cd sachinchaudhary1808.github.io
   bundle exec jekyll serve
   ```

3. Push your changes to GitHub to update the live website.
